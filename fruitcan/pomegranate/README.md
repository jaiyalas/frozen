# Pomegranate

A LaTeX Beamer Theme. [Example slide](www.slideshare.net/godfat/2012-0508lambdadraft).

## Author

 * Yun-Yan Chi a.k.a. Jaiyalas or Eatt

## USAGE

Insert 
```tex
\usetheme{jaifat}
``` 
in your .tex file.

## REQUIREMENT

 * XeTeX
 * fontspec
 * beamer

## FONT REQUIREMENT

 * DejaVu Sans
 * DejaVu Sans Mon
 * DejaVu Serif
 * LiHei Pro
 * Hiragino Maru Gothic Pro W4
